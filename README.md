# Stroboscope
MOSFET으로 LED를 고속 스위칭하여 스트로보 효과를 만드는 장치<br>

# 사용 부품
1 * `Arduino Nano`<br>
1 * N-Channel MOSFET(`IRFZ44N`)<br>
2 * 커패시터(63V, 100μF)<br>
1 * 스텝 업 모듈(7.4V → 10.5V)<br>
2 * 온오프 스위치<br>
1 * 10K 가변저항<br>
1 * 10V ~ 30V LED(48W, 3520LUM)<br>
1 * 2셀(7.4V) 리튬 폴리머 배터리<br>

# 참고사항
업로드를 하려면 아두이노 IDE 라이브러리 폴더에 PWM 라이브러리가 포함되어 있어야 함.<br>
